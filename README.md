# AI Agent CLI - Deepseek 工具型智能体

基于 Vercel AI SDK + TypeScript 实现的完整可交互本地 AI Agent。

开箱即用，内置文件读写、计算器、时间查询工具，自带对话记忆、上下文自动压缩、流式输出交互能力。

## ✨ 项目核心特性

1. 多工具扩展体系

   内置 4 个开箱即用本地工具，遵循 AI SDK 标准 `tool()`开发规范，可无限扩展：

   - `readFile`：读取本地任意文本文件
   - `writeFile`：写入文件（带用户二次确认，防止误覆盖）
   - `calculate`：安全数学计算器（支持开方、幂运算、π/e 常量，输入白名单防注入）
   - `getCurrentTime`：获取国内时区标准日期时间

2. 流式对话交互

   使用 `streamText`实现打字机实时流式输出，工具调用日志可视化打印，交互体验流畅。

3. 对话上下文自动压缩

   限制最大保留 6 条对话记录，超长历史自动调用 LLM 总结压缩，解决长对话上下文溢出问题。

4. 持久化记忆存储

   自动提取对话内用户关键信息，保存至 `agent-memory.txt`，下次启动自动加载记忆注入系统提示词。

5. 安全可控工具调用

   限制单次对话最多 10 轮工具调用，防止无限循环调用；文件写入必须人工确认。

6. 模型低耦合设计

   基于 AI SDK 适配器架构，仅修改一行代码即可切换 OpenAI / Anthropic 等大模型，业务代码无需改动。

7. TypeScript 强类型校验

   使用 Zod 做工具入参运行时校验，参数规范清晰，降低模型调用出错概率。

## 🚀 快速启动

### 1. 前置要求

- Node.js ≥ 18
- Deepseek API Key（前往 [Deepseek 开放平台](https://link.wtturl.cn/?target=https%3A%2F%2Fwww.deepseek.com%2F&scene=im&aid=497858&lang=zh) 申请）

### 2. 安装依赖

```
npm install
```

### 3. 配置环境变量

新建 `.env` 文件，填入你的密钥：

```
DEEPSEEK_API_KEY=你的API密钥
```

### 4. 运行智能体

```
npx tsx hello-agent.ts
```

### 5. 使用交互

启动后进入命令行对话界面：

- 直接输入问题，AI 会自动判断是否调用工具完成任务
- 输入 `exit` 退出程序
- 对话记忆自动保存到 `agent-memory.txt`

## 🧠 核心运行逻辑拆解

1. REPL 交互式循环`readline`实现终端持续问答，维护全局`messages`对话上下文数组。

2. 上下文压缩逻辑

   对话超过 6 轮时，自动调用大模型总结早期对话，仅保留最新交互记录 + 历史摘要。

3. 记忆加载与持久化

   启动读取本地记忆文件注入系统提示；每轮对话结束自动提取用户信息更新记忆文件。

4. 工具调用驱动

   AI SDK 内部自动完成「模型输出工具指令 → 本地执行工具函数 → 结果回传给模型继续生成」闭环，最高限制 10 轮工具调用。

5. 流式输出渲染`fullStream` 分片，区分文本输出、工具调用、工具返回三种日志，优化打印逻辑，解决文字闪烁消失问题。

## 🔄 切换其他大模型

只需修改模型导入与初始化一行代码，其余全部业务逻辑无需改动：

```
// 示例：切换 OpenAI
import { openai } from '@ai-sdk/openai'

const result = streamText({
  model: openai('gpt-3.5-turbo'), // 仅修改此处
  // ...剩余代码完全不变
})
```

## ⚠️ 注意事项

1. 文件操作仅支持**绝对路径**，工具内置强制校验；
2. `agent-memory.txt` 为本地隐私记忆文件，`.gitignore` 已配置自动忽略，不会上传 GitHub；
3. API 密钥请勿硬编码在代码中，统一存放在 `.env` 文件；
4. 计算器内置输入白名单，仅允许基础数学表达式，规避恶意代码注入风险。
